# 技能資料庫使用說明

## 📚 功能說明

這個技能資料庫可以幫助您:
1. 儲存所有技能在不同等級的效果
2. 自動在網頁上顯示技能的詳細資訊
3. 方便維護和更新技能數據

## 🔍 如何獲取技能數據

### 方法 1: 從 questlog.gg 手動查詢
1. 訪問: `https://questlog.gg/aion-2/zh/db/skill/技能ID?level=等級`
2. 例如: `https://questlog.gg/aion-2/zh/db/skill/17750000?level=16`
3. 查看技能效果並手動填入資料庫

### 方法 2: 從遊戲內查看
1. 在遊戲中打開技能面板
2. 查看每個等級的效果
3. 記錄下來填入資料庫

### 方法 3: 使用瀏覽器開發者工具
1. 打開 questlog.gg 的技能頁面
2. 按 F12 打開開發者工具
3. 查看 Network 標籤,找到 API 請求
4. 可能可以批量獲取數據

## 📝 如何添加新技能

在 `skill-database.js` 中添加:

```javascript
"技能ID": {
    name: "技能名稱",
    category: "分類(如:治癒星、破壞星等)",
    levels: {
        1: ["效果1", "效果2", "效果3"],
        2: ["效果1", "效果2", "效果3"],
        // ... 繼續添加所有等級
    }
}
```

## 🎯 範例

已經添加的範例技能:
- **不死帳幕** (ID: 17750000)
  - 分類: 治癒星
  - 等級 16 效果: 
    - 防禦力增加32%
    - 暴擊抵抗增加250

## 🔧 整合到 aion.html

要在您的網頁中使用這個資料庫,需要:

1. 在 `aion.html` 的 `<head>` 中添加:
```html
<script src="skill-database.js"></script>
```

2. 修改技能顯示代碼,添加效果顯示

我可以幫您完成整合步驟!

## 💡 建議的工作流程

1. **先添加常用技能** - 從您角色常用的技能開始
2. **逐步擴充** - 每次添加幾個技能
3. **團隊協作** - 可以和其他玩家一起收集數據
4. **定期更新** - 遊戲更新後檢查技能是否有變化
